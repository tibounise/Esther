# Esther

Image export for micro-controllers with [floyd-steinberg dithering](https://en.wikipedia.org/wiki/Floydâ€“Steinberg_dithering). Compatible with [Adafruit-GFX-Libary](https://github.com/adafruit/Adafruit-GFX-Library) and [SSD1306-AVR](https://github.com/tibounise/SSD1306-AVR) bitmap formats.

## Usage

```
esther myImage.png
```

**Notice :** _only_ PNG images are currently supported.
